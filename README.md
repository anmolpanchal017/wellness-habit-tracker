# Wellness Rhythm — Personal Health Habit Tracker

A **mobile-first**, **offline-friendly** (static) web app for students and young adults to track six daily wellness habits, see a **0–100 Wellness Score**, a **7-day streak calendar**, **weekly charts**, and a **rule-based encouragement tip**.

## 🚀 Features

* Goal setup for daily habits
* Today dashboard with quick inputs
* Wellness score (0–100)
* 7-day streak calendar
* Weekly insights with charts
* Rule-based weekly tips
* Dark mode & settings
* Works fully offline (localStorage)

---

## ⚙️ Setup

```bash
cd health
npm install
npm run dev
```
